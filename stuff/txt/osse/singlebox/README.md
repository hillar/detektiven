# OSSE singlebox
almost all in one single box:
* solr
* elasticsearch
* tika
* etl
* fileserver
* osse

* depends on *[freeipa](../../../../../freeipa)* 

## start

1. mkdir testsinglebox
1. cd testsinglebox
1. wget https://raw.githubusercontent.com/hillar/detektiven/master/stuff/txt/osse/singlebox/Vagrantfile
1. wget https://raw.githubusercontent.com/hillar/detektiven/master/stuff/txt/osse/singlebox/install-OSSE-singlebox.bash
1. bash install-OSSE-singlebox.bash
1. wait ..
1. vagrant up
1. open http://192.168.11.2:9983
