## Requirements

The setup needs about XX GB of free RAM and YY GB disk space.

## Build

```shell
$ cd yourfavouritedirectoryname
$ curl -s https://raw.githubusercontent.com/hillar/detektiven/master/vagans/getit.bash | bash
$ cd detektiven/scripts
$ more getup.bash
$ chmod +x getup.bash
$ time ./getup.bash
```
